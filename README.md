# Identification of Accident-Prone Zones on National Highway 48

Spatial Analysis of Accident-Prone Zones on National Highway 48 using QGIS.

### Project Overview
Is project mein NH-48 (Delhi-Gurugram stretch) par accidents ka spatial analysis kiya gaya hai taaki high-risk zones ko identify kiya ja sake. Ye NIT Durgapur ke internship curriculum ka hissa hai.

###  Methodology
* **Digitization:** NH-48 highway ki centerline ko trace kiya gaya.
* **Buffer Analysis:** Road ke dono taraf **100-meter** ka impact zone (buffer) create kiya gaya.
* **Visualization:** Accident data points ko map par plot karke hotspot clusters identify kiye gaye.

### Tools Used
* QGIS 3.x
* OpenStreetMap (OSM)
* CSV Dataset (Accident Coordinates)
