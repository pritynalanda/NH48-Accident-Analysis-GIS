# Identification of Accident-Prone Zones on National Highway 48

Spatial Analysis of Accident-Prone Zones on National Highway 48 using QGIS.

### Project Overview
This project involves the spatial analysis of road accidents on the NH-48 (Delhi-Gurugram stretch) to identify high-risk zones and hotspots. This work was completed as part of the internship curriculum at National Institute of Technology (NIT), Durgapur.

###  Methodology
* **Digitization:** Traced the centerline of the NH-48 highway to create a precise vector road network.
* **Buffer Analysis:** Created a 100-meter impact zone (buffer) on both sides of the road to analyze the proximity of accidents.
* **Visualization:** Plotted accident data points using coordinate geometry to identify and visualize significant hotspot clusters.

### Tools Used
* **GIS Software:** QGIS 3.x
* **Basemaps:** OpenStreetMap (OSM)
* **Data Handling:** CSV Dataset (Accident Coordinates)
